# Honeypot Lab using Pentbox

## Overview
This project demonstrates a basic honeypot setup using Pentbox in a controlled lab environment.

## Objective
To understand how honeypots capture and log interaction attempts.

## Environment
- Kali Linux (VirtualBox)
- Pentbox

## Steps Performed
- Installed and launched Pentbox
- Configured honeypot on custom ports
- Simulated traffic within local environment
- Captured logs of incoming connections

## Observations
- Detected interaction attempts
- Captured HTTP requests (GET / HTTP/1.1)
- Logged User-Agent and request headers

## Sample Log
INTRUSION ATTEMPT DETECTED  
GET / HTTP/1.1  
User-Agent: Mozilla/5.0  

## Learning Outcome
This project helped in understanding:
- Basic honeypot concepts  
- Log capturing and analysis  
- How interaction attempts are recorded  

## Note
This was performed in a controlled lab environment for learning purposes.
